# PIXIS-CODE
Drivers and Utilities for the PIXIS Project

Select the appropriate driver for Landscape (st7789vl.bin) or Portrait (st7789vp.bin) screen orientation and place the LCD binary driver in your /usr/lib/firmware directory. If you are using an OS other than Raspberry Pi then copy the driver into the appropriate directory. 

Follow the instructions in LCDconfigs.txt and edit your config.txt file.
